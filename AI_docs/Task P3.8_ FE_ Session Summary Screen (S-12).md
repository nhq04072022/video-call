# Task P3.8: FE: Session Summary Screen (S-12)

**Mô Tả:** Xây dựng màn hình Tóm tắt Phiên sau khi phiên kết thúc.

**Team:** Frontend
**Ước Tính:** 1.0 Person-Day

## Công Việc Cần Thực Hiện

1.  **Xây dựng UI Tóm tắt (S-12):** Thiết kế màn hình đơn giản hiển thị thông tin tóm tắt phiên.
2.  **Hiển thị Thông tin:**
    *   Hiển thị thời lượng phiên (lấy từ API sau khi phiên kết thúc).
    *   Hiển thị tên Mentor/Mentee.
    *   Các nút hành động tiếp theo (ví dụ: "Quay lại Dashboard").
3.  **Tích hợp API:** Lấy dữ liệu tóm tắt từ API `/sessions/:sessionId` (P3.1).

## Tiêu Chí Hoàn Thành
*   Màn hình Tóm tắt hiển thị chính xác thời lượng phiên.
*   Người dùng được chuyển hướng đến màn hình này sau khi phiên kết thúc.
