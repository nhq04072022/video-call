# Task P1.1: Setup Project (BE/FE)

**Mô Tả:** Thiết lập môi trường phát triển, kho lưu trữ mã nguồn (Git/GitHub), và hệ thống Tích hợp/Triển khai Liên tục (CI/CD) cơ bản cho cả Frontend và Backend.

**Team:** BE/FE
**Ước Tính:** 1.0 Person-Day

## Công Việc Cần Thực Hiện

### Backend (BE)
1.  Khởi tạo dự án Backend (Node.js/Python/Go, tùy chọn).
2.  Cấu hình môi trường phát triển (Development Environment) và môi trường kiểm thử (Testing Environment).
3.  Thiết lập kết nối cơ bản với PostgreSQL (chưa cần tạo schema).
4.  Cấu hình linter và formatter (ví dụ: ESLint/Prettier).

### Frontend (FE)
1.  Khởi tạo dự án Frontend (Next.js/React, TypeScript).
2.  Cấu hình TypeScript ở chế độ nghiêm ngặt (`strict mode`).
3.  Thiết lập TailwindCSS và cấu hình các token thiết kế cơ bản (màu sắc, font).
4.  Cấu hình linter và formatter.

### Chung
1.  Tạo kho lưu trữ Git và thiết lập cấu trúc thư mục chung.
2.  Thiết lập CI/CD cơ bản (ví dụ: chạy linter/tests khi push code).

## Tiêu Chí Hoàn Thành
*   Cả hai dự án FE và BE đều chạy được trên máy local.
*   Linter và formatter được cấu hình và chạy thành công.
*   Các token màu sắc chính (Primary Purple, Text Grey) đã được định nghĩa trong Tailwind config.
