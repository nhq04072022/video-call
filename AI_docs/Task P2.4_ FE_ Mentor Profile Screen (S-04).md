# Task P2.4: FE: Mentor Profile Screen (S-04)

**Mô Tả:** Xây dựng giao diện Hồ sơ Mentor chi tiết, bao gồm các tab và nút "Apply".

**Team:** Frontend
**Ước Tính:** 2.0 Person-Days

## Công Việc Cần Thực Hiện

1.  **Xây dựng UI Hồ sơ (S-04):** Thiết kế bố cục trang Hồ sơ Mentor theo TDD.
2.  **Tab "About Me":** Hiển thị chi tiết thông tin Mentor (bio, experience, achievements) lấy từ API P2.1.
3.  **Tab "Sessions":** Hiển thị lịch sử các phiên đã hoàn thành (chỉ cần hiển thị danh sách, chưa cần chi tiết).
4.  **Nút "Apply" (FR-2.4):** Đặt nút hành động chính, khi click sẽ kích hoạt Modal/Chuyển hướng đến Form Đăng ký Phiên (S-05).
5.  **Tích hợp API:** Sử dụng API `GET /profiles/:userId` (P2.1) để lấy dữ liệu hiển thị.

## Tiêu Chí Hoàn Thành
*   Trang Hồ sơ Mentor hiển thị đầy đủ thông tin và có thiết kế tương tự Menteelogy.
*   Nút "Apply" hoạt động và chuyển hướng đúng.
*   Dữ liệu được tải thành công từ Backend.
